Clipboard Boss

Java application to manage a system clipboard history

Installer 1.0 .jar Last Source .zip View on GitHub

Clipboard Boss is an utility for manage the clipboard history

The project is free, crossplatform, multilanguage and it's somehow regularly updated and supported.

It’s simple little fast program that non impact the system performances.

Completely Free, no spyware, no ads and no user tracking.

Platforms

Runs on all platforms: Windows, Linux, Mac.
Currently tested on: Windows 7, Windows 8, Linux Ubuntu 14.04, Mac OS X.

Languages

Translated in four language: English, Italian, French, German.

Requirements

Only requires Java SE 6+

Install

Launching installer is easy. On most operating systems, a double-click on the "ClipboardBoss-uni.jar" file icon will do.
Otherwise, you may need to:
launch with a right-click on the "ClipboardBoss-uni.jar" file icon and select Open with Java Runtime
launch from a shell command: java -jar ClipboardBoss-uni.jar
